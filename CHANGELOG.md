# История изменений GOART

Формат основан на идее понятных пользовательских и технических изменений по версиям.

## v0.1.1 — 2026-06-13

Статус: документационный и стабилизационный патч.

### Добавлено

- Добавлен полноценный `README.md` вместо краткого описания.
- Добавлена папка `docs/`.
- Добавлен обзор проекта `docs/PROJECT_OVERVIEW.md`.
- Добавлена архитектурная документация `docs/ARCHITECTURE.md`.
- Добавлен документ игрового дизайна `docs/GAME_DESIGN.md`.
- Добавлен список известных ошибок `docs/KNOWN_BUGS.md`.
- Добавлена дорожная карта `docs/ROADMAP.md`.
- Добавлен файл версии `version.json`.

### Найдено

- `resetGame()` обращается к отсутствующему элементу `#player-name`.
- `shuffle()` объявлена дважды.
- `legendOpen()` может удалять кнопку закрытия справки.
- Нужно проверить наличие файлов портретов NPC.

### План исправления

- Исправить запуск игры без ошибки `null.innerText`.
- Добавить отображение версии в интерфейс и заголовок страницы.
- Исправить окно справки.
- Удалить дублирование `shuffle()`.
- После проверки создать стабильную точку отката.
