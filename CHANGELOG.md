# История изменений GOART

Формат основан на идее понятных пользовательских и технических изменений по версиям.

## v0.1.3 — 2026-06-13

Статус: реальный патч запуска.

### Добавлено

- Добавлен файл `js/startup-fix-v0.1.3.js`.
- Патч подключён в `index.html` после основного `main.js`.
- Версия `v0.1.3` показана в `<title>` страницы.
- Версия `v0.1.3` показана в шапке интерфейса.

### Исправлено

- Исправлено падение игры на старте из-за обращения к отсутствующему элементу `#player-name`.
- Добавлена безопасная замена `resetGame()`.
- Добавлена безопасная замена `legendOpen()`, которая сохраняет кнопку закрытия справки.

### Важно

- Основной `main.js` пока не переписан напрямую.
- Патч подключается отдельным файлом, чтобы минимально рисковать сломать большой основной файл.
- Следующий этап — проверить запуск на GitHub Pages и затем аккуратно перенести исправления внутрь `main.js`.

## v0.1.2 — 2026-06-13

Статус: патч версионирования и стабилизации запуска.

### Добавлено

- Обновлён `version.json` до версии `0.1.2`.
- Зафиксировано правило: каждый следующий патч должен иметь понятное описание изменений.
- Подготовлено отображение версии в интерфейсе и заголовке страницы.

### Исправляется

- Ошибка запуска `resetGame()`, связанная с обращением к отсутствующему DOM-элементу `#player-name`.

### Планируется в этом патче

- Показать `v0.1.2` в `<title>` страницы.
- Показать `v0.1.2` в шапке интерфейса.
- Исправить `legendOpen()`, чтобы не исчезала кнопка закрытия справки.

## v0.1.1 — 2026-06-13

Статус: документационный и стабилизационный патч.

### Добавлено

- Добавлен полноценный `README.md` вместо краткого описания.
- Добавлена папка `docs/`.
- Добавлен обзор проекта `docs/PROJECT_OVERVIEW.md`.
- Добавлена архитектурная документация `docs/ARCHITECTURE.md`.
- Добавлен документ игрового дизайна `docs/GAME_DESIGN.md`.
- Добавлен список известных ошибок `docs/KNOWN_BUGS.md`.
- Добавлена дорожная карта `docs/ROADMAP.md`.
- Добавлен файл версии `version.json`.

### Найдено

- `resetGame()` обращается к отсутствующему элементу `#player-name`.
- `shuffle()` объявлена дважды.
- `legendOpen()` может удалять кнопку закрытия справки.
- Нужно проверить наличие файлов портретов NPC.

### План исправления

- Исправить запуск игры без ошибки `null.innerText`.
- Добавить отображение версии в интерфейс и заголовок страницы.
- Исправить окно справки.
- Удалить дублирование `shuffle()`.
- После проверки создать стабильную точку отката.
