# Игровой дизайн GOART

## Жанр

GOART — небольшая браузерная игра-симулятор с элементами квеста, интерактивной карты и юмористической рабочей истории.

## Сеттинг

Место действия — типография.

Игрок перемещается между офисом, коридорами, складом, аквариумом, производством, лаком и кабинетом босса.

## Игрок

Игрок начинает игру как сотрудник типографии.

Основные параметры:

- имя;
- текущая локация;
- стресс;
- инвентарь;
- состояние квестов;
- флаг завершения игры.

## Основная игровая петля

1. Игрок выбирает соседнюю комнату.
2. Система проверяет лимит помещения.
3. Игрок перемещается.
4. Проверяется наличие NPC в той же комнате.
5. При встрече запускается событие, диалог или действие.
6. Игрок получает предмет, теряет предмет, снижает или повышает стресс.
7. Игрок выполняет квесты.
8. После выполнения основных задач открывается путь к финалу.

## Квестовая цепочка

### 1. Цветопроба

Связанный NPC: Катя.

Игрок должен получить задание и сделать цветопробу.

### 2. Лак

Связанный NPC: Антон / локация «Лак».

После цветопробы игрок должен выполнить задачу с лаком.

### 3. Финал

Связанный NPC: Виктор.

После выполнения цветопробы и лака игрок может пройти финальный блиц.

## Типы NPC

Текущие типы используются как игровые роли:

- `kvest` — персонаж, связанный с квестом;
- `barrier` — препятствие;
- `neutral` — нейтральный персонаж;
- `tip` — персонаж-подсказка;
- `happy` — снижает стресс;
- `take` — может отнять предметы;
- `help` — помогает игроку;
- `boss` — финальный персонаж.

## Стресс

Стресс — простой показатель состояния игрока.

Некоторые NPC увеличивают стресс, другие уменьшают.

В будущем стресс можно использовать для:

- проигрыша при достижении 100;
- изменения вариантов диалогов;
- ограничения действий;
- дополнительной концовки.

## Финальный блиц

Босс Виктор запускает мини-викторину по печатной тематике.

Сейчас блиц содержит вопросы про:

- CMYK;
- Pantone;
- подготовку макета к печати;
- допечатную подготовку;
- PDF;
- плашечные цвета.

Игрок должен ответить на серию вопросов за ограниченное время.
