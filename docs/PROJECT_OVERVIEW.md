# GOART — обзор проекта

## Назначение

GOART — браузерная игра на русском языке с рабочим названием «Рабочий день в типографии».

Игрок перемещается по локациям типографии, встречает NPC, выполняет задания и проходит рабочий день до финального события с боссом.

Проект ориентирован на простой стек:

- HTML;
- CSS;
- Vanilla JavaScript;
- GitHub Pages;
- без backend-части.

## Главная цель проекта

Довести GOART до полностью рабочего состояния:

1. игра запускается без ошибок;
2. карта корректно отображается;
3. NPC появляются и перемещаются;
4. диалоги открываются и закрываются;
5. квесты проходятся от начала до финала;
6. проект стабильно работает на мобильном экране;
7. код и документация понятны для дальнейшего развития.

## Текущая игровая концепция

Игра моделирует рабочий день в типографии.

Внутри проекта уже есть:

- карта из 13 локаций;
- NPC с ролями и диалогами;
- квест цветопробы;
- квест лака;
- финальный босс;
- блиц-вопросник по теме печати и допечатной подготовки;
- стресс игрока;
- инвентарь;
- ограничения вместимости комнат.

## Основные сущности

### Room

Локация на карте. Описывает место, куда может попасть игрок или NPC.

Ключевые поля:

- `id`;
- `name`;
- `icon`;
- `doors`;
- `desc`;
- `limit`.

### NPC

Персонаж, который появляется на карте, может двигаться, запускать события, помогать или мешать игроку.

Ключевые поля:

- `name`;
- `icon`;
- `portrait`;
- `desc`;
- `type`;
- `home`;
- `workplace`;
- `spawn`;
- `moveInterval`;
- `dialog1`, `dialog2`, `dialog3`;
- `prob1`, `prob2`, `prob3`.

### Quest

Задание, которое игрок должен выполнить для прохождения игры.

Сейчас есть три квеста:

- `proba`;
- `lak`;
- `boss`.

### GameState

Глобальное состояние игры. Включает игрока, NPC, диалоги и прогресс прохождения.

## Связь с AWB

AWB — отдельная идея Android WebView-приложения, которое может открывать GOART или другие сайты.

AWB не является частью основной игры, но может стать способом запуска GOART на телефонах в управляемом режиме.

Документ с историческими заметками: `PROJECT_NOTES_AWB_GOART.md`.
