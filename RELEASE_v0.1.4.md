# GOART v0.1.4

Дата: 2026-06-13

Тип версии: runtime-startup-fix.

## Назначение версии

Версия `v0.1.4` исправляет важную логическую проблему предыдущего патча `v0.1.3`.

`v0.1.3` добавил безопасный `resetGame()`, но основной `main.js` всё ещё назначал старый `window.onload`. Из-за этого старый небезопасный запуск мог сработать раньше патча.

## Что изменено

- Добавлен файл `js/runtime-fix-v0.1.4.js`.
- `index.html` теперь подключает `js/runtime-fix-v0.1.4.js` вместо `js/startup-fix-v0.1.3.js`.
- Старый `window.onload` из `main.js` отключается после загрузки `main.js`.
- Автозапуск игры перенесён в безопасную функцию `bootGame()`.
- В `bootGame()` выставляется версия, скрывается `eventBox` и запускается безопасный `resetGame()`.
- Обновлены `version.json` и `CHANGELOG.md`.

## Что проверить

1. При открытии страницы появляется запрос имени.
2. После ввода имени карта появляется без ошибки `null.innerText`.
3. В шапке отображается `GOART v0.1.4`.
4. В title вкладки отображается `GOART v0.1.4`.
5. Работает справка.
6. Справка закрывается.
7. Кнопки перемещения появляются внизу.
